# CatFeatures

Котовые фитчи.
- Генерация идентификаторов

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cat-features'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cat-features
